# ASL-Glove

pip install pyserial

### Active Virtual Environment
mac: source venv/bin/activate
Windows: .venv\Scripts\Activate

### Instructions
- Make sure HC-05 is paired to device and the arduino is powered, change BLUETOOTH_PORT value in gloveDisplay.py to either COM6 or /dev/cu.HC-05 (Depends on if device is a windows or mac). 
- in terminal run python gloveDisplay.py
- Click Connect to ASL Glove button
